Projetos

BakaComic

O projeto BakaKomik foi uma jornada inspiradora, baseada no design de uma interface de um site para leitura de mangás. Foi um desafio emocionante, especialmente porque estava no início da minha jornada como desenvolvedor web e ainda estava explorando a estruturação de sites. Decidi transformar o incrível design que encontrei online em uma realidade utilizando ferramentas como HTML, CSS, Bootstrap, SASS e Gulp, todas as quais eu estava estudando na época.

O design original, criado pela talentosa designer Panda Design (você pode conferir o design original e o perfil dela aqui: Link para o Design Original), desafiou minha habilidade de manter a fidelidade das proporções e a precisão no desenvolvimento, considerando que só tinha acesso à imagem do projeto. No entanto, a determinação prevaleceu e superei esse desafio.

Essa experiência me proporcionou não apenas um profundo entendimento da análise de design e criatividade para adaptar designs em websites, mas também a habilidade de garantir precisão e responsividade em projetos futuros, especialmente quando se trata de tamanhos e layouts.

Relogio digital

O projeto Relógio Digital se destaca como um dos meus favoritos, pois sua combinação de design e funcionalidade foi um resultado gratificante das habilidades que adquiri ao longo do tempo. Desenvolvido com HTML, CSS e JavaScript puro, a proposta era criar um relógio digital atraente visualmente.

Este projeto se destacou não apenas pelo visual, mas também por sua complexa estrutura. Ele consiste em duas telas intercambiáveis por meio de um interruptor. A primeira tela apresenta um relógio digital e um cronômetro altamente preciso, enquanto a segunda tela oferece um calculador de datas que permite calcular quanto tempo falta para o seu próximo aniversário.

Embora tenha sido um desafio, já que envolveu um nível significativo de JavaScript, consegui superar essa etapa e aprimorar minhas habilidades de programação, lógica e resolução de problemas. Este projeto representa meu comprometimento em transformar ideias criativas em realidade, além de contribuir para meu desenvolvimento profissional.

Cinematic Nightmare

O projeto Cinematic Nightmare nasceu como uma plataforma dedicada a entusiastas de filmes de terror, oferecendo recomendações cuidadosamente selecionadas. O site foi concebido em duas páginas distintas e complementares.

Na primeira página, apresentamos os filmes de terror em um carrossel dinâmico, exibindo o título, uma breve descrição e imagens representativas. A interatividade ganha vida à medida que você navega pelas imagens, alterando instantaneamente as informações exibidas.

A segunda página mergulha mais fundo em cada filme, proporcionando detalhes como título, descrição expandida e a inclusão de trailers. A navegação é facilitada por meio dos links no menu de navegação do site.

Toda essa experiência foi desenvolvida com HTML, CSS e JavaScript puro, e a integração dos vídeos do YouTube enriqueceu a interação com o conteúdo. O Cinematic Nightmare representa a combinação de paixão por filmes de terror e habilidades técnicas, criando um espaço cativante para amantes desse gênero cinematográfico

Five leaf clover

O projeto Five Leaf Clover ocupa um lugar especial em minha jornada, sendo meu primeiro projeto de grande complexidade e tamanho. Como amante de vinhos, esta loja virtual de vinhos foi um desafio emocionante e uma realização gratificante.

A loja apresenta uma estrutura complexa, com seções dedicadas a diferentes tipos de vinhos, como Tintos, Brancos, Rosés, Orgânicos e até vinhos enlatados. Além disso, a categorização por país tornou a experiência do usuário ainda mais envolvente.

O projeto foi desenvolvido principalmente com HTML, com adições de CSS para estilização e uma pitada da biblioteca jQuery para validação de formulários no rodapé do site.

Five Leaf Clover representa não apenas minha paixão por vinhos, mas também um marco em minha carreira, marcando meu progresso e aprendizado na criação de projetos complexos.

Tempo agora

O projeto Tempo Agora marcou uma virada em minha jornada, sendo meu primeiro projeto com o uso do Vue.js e a integração de uma API de previsão do tempo. Foi um desafio significativo que testou diversas habilidades.

Trabalhando com Vue, deparei-me com a complexidade da separação de componentes e o uso de props para a transferência de dados entre eles. Além disso, a lógica de programação desempenhou um papel vital na criação de uma estrutura de código limpa e eficiente.

Este projeto foi uma experiência intensiva em Vue, representando mais de 60% do desenvolvimento, com toques de CSS e JavaScript para complementar a funcionalidade do Vue. Além disso, integrei uma API de imagens geradas aleatoriamente com base em palavras-chave, aprimorando ainda mais a experiência do usuário. Tempo Agora não apenas expandiu meu conjunto de habilidades, mas também marcou meu comprometimento em criar soluções tecnológicas eficazes

IMC Plus

O projeto IMC Plus representa um marco em minha carreira, sendo minha primeira incursão no mundo do React.js. Este projeto consiste em duas telas essenciais.

Na primeira tela, desenvolvi um formulário interativo que coleta informações como nome, idade, altura, peso, entre outros. Ao clicar em 'Cadastrar', o site calcula automaticamente o Índice de Massa Corporal (IMC) e prepara os dados para a próxima tela.

A segunda tela exibe todas as informações inseridas no formulário e inclui uma representação visual do corpo humano, ajustada de acordo com a seleção de gênero. Além disso, nesta segunda tela, você encontrará uma tabela detalhada com o IMC, o estado do corpo e uma descrição correspondente.

Tendo experiência prévia com Vue.js, a transição para React foi mais suave devido à semelhança na estrutura de componentização. No entanto, esta experiência me permitiu aprofundar meu conhecimento em styled-components, uma abordagem que permite componentizar o CSS de forma individual e global. IMC Plus não apenas expandiu minha capacidade técnica, mas também adicionou um valioso conjunto de habilidades em minha jornada de desenvolvimento
